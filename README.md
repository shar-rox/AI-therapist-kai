# AI-therapist-kai

Short description
A simple TypeScript-based AI assistant prototype that simulates therapist-like conversation flows (demo/proof-of-concept). This project demonstrates using an LLM backend and a small typed codebase to manage conversation state and prompts.

## Quickstart (developer)
1. Clone:
   git clone https://github.com/shar-rox/AI-therapist-kai
2. Install:
   cd AI-therapist-kai
   npm install
3. Configure environment:
   - Copy `.env.example` to `.env` and fill in API keys if required.
4. Run (dev):
   npm run dev
5. Run tests:
   npm test
